This project analyzes employee base salary data from a fictional healthcare company, Acorn Health Systems, to understand how compensation varies by role and department using SQL.

I examined salary data to identify which roles are paid the most within each department and which role shows the greatest pay variability across the organization. The analysis focuses on using SQL aggregations, window functions, and basic statistical measures to surface insights that support compensation benchmarking and pay equity discussions.
